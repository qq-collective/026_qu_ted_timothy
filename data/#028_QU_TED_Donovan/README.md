# QU_TED_Donovan｜#028

## これは何？

大学ゼミの英語発表後の議論を記録し、
「その場の動き」を可視化・アーカイブするための2段階ツールです。

QU_TEDシリーズの第三弾。
Eliceが整えた「声の地図（初期ストック）」を受け取り、
議論の推移（フロー）を記録して、
場がどう変容したかを最終的なアーカイブ（ストック）として残します。

---

## なぜ作ったか

Eliceが発表直後の「静止画」を撮ります。
Donovanはその後の「動画」を回します。

マトリックスが出た。でもそこから誰も動かない——という場面があります。
「何を話せばいいかわからない」ではなく、「どこから入ればいいかわからない」が原因です。

Donovanは議論の入口を3つ用意します。
答えを求めない、意見が分かれそうな、その場の誰かが話せそうな問いを選んで投げる。

フリートークが終わったら、ZoomやTeamsの文字起こしを受け取り、
今日起きたことをアーカイバーの目線で記録します。
その場で英語が出なくても「あ、こう言えばよかったんだ」という事後の学びが残ります。

---

## できること

**LIVE フェーズ**
- Eliceのマトリックスから議論の火種を3つ生成（日本語＋英語セット）
- スクリーンに表示してフリートークのきっかけに

**ARCHIVE フェーズ**
- ZoomやTeamsの文字起こしをそのまま貼り付けて処理
- セッションを場面ごとに描写（OPENING / TENSION / TURNING POINT など）
- 印象的な発言を引用＋英語訳で収録
- Today's Takeaway（今日一番大事なこと）を1〜2文で抽出
- 次回への問いを英語3問で生成
- 文字起こしがない場合もマトリックスと火種から推測して記録を生成

---

## 使い方

1. ブラウザでHTMLファイルを開く
2. APIキーを入力（ゼミ運営側で管理推奨）
3. お題＋EliceのマトリックスをコピペしてLIVEスタート
4. 議論の火種3つがスクリーンに表示される
5. フリートーク中はZoom/Teamsの文字起こし機能をオンにしておく
6. 終わったら文字起こしをそのまま貼り付けてARCHIVEボタン
